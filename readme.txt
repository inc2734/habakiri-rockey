== Description ==

Rocky is a design skin of Habakiri. This plugin needs Habakiri 2.0.0 or later.

== Installation ==

1. Upload `Rocky` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
